איך נראה יום העבודה שלך?

זה יקבע את ה-layout ואת מה שמופיע ראשון כשנפתח את הטרמינל

A

סריקת שוק רחבה → צלילה → פעולה

פותח את הטרמינל, רואה את כל 4,050 החדרים, מחפש הזדמנויות CALL/PUT, בוחר חדר ספציפי, מנתח, מבצע override או opportunity

UI מתאים: טבלת סיגנלים גדולה במרכז, chart נפתח בלחיצה על שורה
B

מלון ספציפי → חדרים → פעולה

כבר יודע איזה מלון מעניין אותי, נכנס ישר אליו, בודק את כל החדרים והתאריכים, מחליט מה לעשות

UI מתאים: Hotel selector בולט, drilldown מהיר לחדרים
C

אלרטים ← מה השתנה? → חקירה → פעולה

פותח את הטרמינל ורואה מה השתנה מהפעם הקודמת — סיגנלים חדשים, signal flips, price drops, voter changes — ואז חוקר

UI מתאים: Alert feed + "What Changed" panel בראש, טבלה משנית
D

משלב — תלוי ביום

לפעמים סריקה רחבה, לפעמים אלרטים, לפעמים מלון ספציפי. צריך שכל הגישות יהיו נגישות מאותו מסך

UI מתאים: Mode switcher עם כניסה מהירה לכל workflow
